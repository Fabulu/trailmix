ENEMY SPRITE ART BRIEF V2 — Trail Mix
=========================================
Top-down roguelike, 256x192 NDS. Clean pixel art, simple geometric-creature style.

FORMAT RULES (read these first)
-------------------------------
- 4bpp indexed color: 16 colors per sprite, index 0 = transparent
- NO anti-aliasing, NO dithering
- 1px dark outline on all bodies (darkest color in the palette, never pure black #000)
- Horizontal strip layout: frames are placed left-to-right in a single row
- Transparent background (index 0)
- File format: PNG, will be converted to linear 4bpp binary
- Each enemy type has a distinct dominant hue readable even at 8x8 pixels
- Bosses should look like larger, more dangerous creatures — NOT scaled-up regulars
- High-contrast bodies against transparent BG. Rich, saturated palette.

SIZE GUIDE
----------
- Small  (s): one frame = 8x8 px.   2-frame strip = 16x8 PNG
- Medium (m): one frame = 16x16 px.  2-frame strip = 32x16 PNG
- Large  (l): one frame = 24x24 px.  2-frame strip = 48x24 PNG
- Boss:       one frame = 32x32 px.  2-frame strip = 64x32 PNG
- Boss Large: one frame = 64x64 px.  8-frame strip = 512x64 PNG

SIZE DIFFERENCES: Small = minimal detail, just silhouette + dominant color.
Medium = full detail, readable features. Large = same character, bulkier
proportions, extra detail (scars, armor, thicker limbs).

DIRECTORY: All source PNGs go in graphics/enemies/
Converted .bin files go in arm9/data/enemies/ (handled by build pipeline)

22. THE APOTHECARY — Final Boss (Wave 30)
    Dominant colors: WHITE LAB COAT, GOLD mortar-and-pestle weapon, shifting
    accent colors (one per pill color: red, blue, green, yellow, purple, orange).

    A towering alchemist figure — the mastermind behind the enemy pill creatures.
    Wears a tattered white coat stained with colorful chemical splashes. Wields
    an oversized golden pestle as both weapon and catalyst. Has a belt of
    bubbling vials. Face obscured by round goggles and a mask. Imposing but
    scientific — a mad pharmacist, not a warrior.

    This is the final boss. It should look like the most important, most
    dangerous, most detailed creature in the entire game. Every pixel counts.

    8 FRAMES (512x64 horizontal strip):
      Frame 0: Idle stance — standing tall, pestle at rest, coat flowing.
      Frame 1: Pestle wind-up — pulling pestle back over shoulder.
      Frame 2: Pestle swing — mid-strike, pestle blurred forward arc.
      Frame 3: Casting — free hand extended, vials glowing, energy gathering.
      Frame 4: Spawning adds — coat flared open, creatures emerging from belt.
      Frame 5: Damage taken — recoiling, goggles cracked, coat torn further.
      Frame 6: Meltdown phase — body glowing, coat dissolving, pestle cracking.
      Frame 7: Death crumble — falling apart, vials shattering, golden dust.

    File: boss_apothecary.png (512x64)
